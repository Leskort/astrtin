║  БЫСТРАЯ ИНСТРУКЦИЯ: Supabase Storage (альтернатива)        ║
╚═══════════════════════════════════════════════════════════════╝

1. СОЗДАЙТЕ ПРОЕКТ В SUPABASE:
   ─────────────────────────────────────────────────────────────
   • Зайдите на https://supabase.com
   • Войдите через GitHub или создайте аккаунт
   • Нажмите "New Project"
   • Имя: astrinn-photos
   • План: Free (бесплатно)
   • Подождите 1-2 минуты

2. СОЗДАЙТЕ STORAGE BUCKET:
   ─────────────────────────────────────────────────────────────
   • В меню проекта: Storage → New bucket
   • Name: astrinn-photos
   • Public bucket: ВКЛЮЧИТЕ ✅
   • Create bucket

3. ПОЛУЧИТЕ КЛЮЧИ:
   ─────────────────────────────────────────────────────────────
   • Settings → API
   • Скопируйте "Project URL"
   • Найдите "service_role" ключ (НЕ anon!)
   • Покажите его (иконка глаза) и скопируйте

4. ДОБАВЬТЕ В NETLIFY:
   ─────────────────────────────────────────────────────────────
   • Site settings → Environment variables
   
   Первая переменная:
   Key:   NEXT_PUBLIC_SUPABASE_URL
   Value: ваш Project URL

   Вторая переменная:
   Key:   SUPABASE_SERVICE_ROLE_KEY
   Value: ваш service_role ключ

5. ПЕРЕСОБЕРИТЕ:
   ─────────────────────────────────────────────────────────────
   • Deploys → Trigger deploy → Deploy site

Подробная инструкция: SUPABASE_SETUP.md

Преимущества:
✅ Работает во многих странах
✅ Бесплатный план: 1GB хранилища
✅ Быстрая загрузка
