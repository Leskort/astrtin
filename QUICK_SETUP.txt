1. ПОЛУЧИТЕ КЛЮЧИ ИЗ CLOUDINARY:
   ─────────────────────────────────────────────────────────────
   • Зайдите на https://cloudinary.com
   • Зарегистрируйтесь (бесплатно)
   • В Dashboard найдите раздел "Account Details"
   • Скопируйте 3 значения:
     - Cloud Name
     - API Key  
     - API Secret

2. ДОБАВЬТЕ В NETLIFY:
   ─────────────────────────────────────────────────────────────
   • Откройте https://app.netlify.com
   • Выберите ваш сайт "astrinn"
   • Site settings → Environment variables
   • Нажмите "Add a variable" 3 раза:

   Первая переменная:
   Key:   CLOUDINARY_CLOUD_NAME
   Value: ваш-cloud-name

   Вторая переменная:
   Key:   CLOUDINARY_API_KEY
   Value: ваш-api-key

   Третья переменная:
   Key:   CLOUDINARY_API_SECRET
   Value: ваш-api-secret

3. ПЕРЕСОБЕРИТЕ:
   ─────────────────────────────────────────────────────────────
   • Deploys → Trigger deploy → Deploy site
   • Или сделайте commit + push в GitHub

4. ПРОВЕРЬТЕ:
   ─────────────────────────────────────────────────────────────
   • Откройте сайт
   • Попробуйте загрузить фотографию
   • Должно работать без ошибок!
